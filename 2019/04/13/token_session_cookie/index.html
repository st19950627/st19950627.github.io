TokenгҖҒSessionгҖҒCookieдёүиҖ…

еүҚеҗҺз«ҜеҲҶзҰ»иә«д»ҪйӘҢиҜҒзҡ„дёүз§ҚжЁЎејҸ

Token

д»ҘдёӢеҮ зӮ№зү№жҖ§дјҡи®©дҪ еңЁзЁӢеәҸдёӯдҪҝз”ЁеҹәдәҺTokenзҡ„иә«д»ҪйӘҢиҜҒ

  1. ж— зҠ¶жҖҒгҖҒеҸҜжү©еұ•

  2. ж”ҜжҢҒз§»еҠЁи®ҫеӨҮ

  3. и·ЁзЁӢеәҸи°ғз”Ё

  4. е®үе…Ё

еҹәдәҺTokenзҡ„иә«д»ҪйӘҢиҜҒзҡ„иҝҮзЁӢеҰӮдёӢ:

  1. з”ЁжҲ·йҖҡиҝҮз”ЁжҲ·еҗҚе’ҢеҜҶз ҒеҸ‘йҖҒиҜ·жұӮгҖӮ

  2. зЁӢеәҸйӘҢиҜҒгҖӮ

  3. зЁӢеәҸиҝ”еӣһдёҖдёӘзӯҫеҗҚзҡ„token з»ҷе®ўжҲ·з«ҜгҖӮ

  4. е®ўжҲ·з«ҜеӮЁеӯҳtoken,е№¶дё”жҜҸж¬Ўз”ЁдәҺжҜҸж¬ЎеҸ‘йҖҒиҜ·жұӮгҖӮ

  5. жңҚеҠЎз«ҜйӘҢиҜҒtokenе№¶иҝ”еӣһж•°жҚ®гҖӮ

жҜҸдёҖж¬ЎиҜ·жұӮйғҪйңҖиҰҒtokenгҖӮtokenеә”иҜҘеңЁHTTPзҡ„еӨҙйғЁеҸ‘йҖҒд»ҺиҖҢдҝқиҜҒдәҶHttpиҜ·жұӮж— зҠ¶жҖҒгҖӮ

иҜ·жұӮдёӯеҸ‘йҖҒtokenиҖҢдёҚеҶҚжҳҜеҸ‘йҖҒcookieиғҪеӨҹйҳІжӯўCSRF(и·Ёз«ҷиҜ·жұӮдјӘйҖ )гҖӮеҚідҪҝеңЁе®ўжҲ·з«ҜдҪҝз”ЁcookieеӯҳеӮЁtokenпјҢcookieд№ҹд»…д»…жҳҜдёҖдёӘеӯҳеӮЁжңәеҲ¶иҖҢдёҚжҳҜз”ЁдәҺи®ӨиҜҒгҖӮдёҚе°ҶдҝЎжҒҜеӯҳеӮЁеңЁSessionдёӯпјҢи®©жҲ‘д»¬е°‘дәҶеҜ№sessionж“ҚдҪңгҖӮ

жңҖз®ҖеҚ•зҡ„tokenз»„жҲҗ:uid(з”ЁжҲ·е”ҜдёҖзҡ„иә«д»Ҫж ҮиҜҶ)гҖҒtime(еҪ“еүҚж—¶й—ҙзҡ„ж—¶й—ҙжҲі)гҖҒsign(зӯҫеҗҚпјҢз”ұtokenзҡ„еүҚеҮ дҪҚ+зӣҗд»Ҙе“ҲеёҢз®—жі•еҺӢзј©жҲҗдёҖе®ҡй•ҝзҡ„еҚҒе…ӯиҝӣеҲ¶еӯ—з¬ҰдёІпјҢеҸҜд»ҘйҳІжӯўжҒ¶ж„Ҹз¬¬дёүж–№жӢјжҺҘtokenиҜ·жұӮжңҚеҠЎеҷЁ)гҖӮиҝҳеҸҜд»ҘжҠҠдёҚеҸҳзҡ„еҸӮж•°д№ҹж”ҫиҝӣtokenпјҢйҒҝе…ҚеӨҡж¬ЎжҹҘеә“

Session

  • sessionзҡ„иө·еӣ пјҡеӣ дёәHTTPиҜ·жұӮжҳҜж— зҠ¶жҖҒзҡ„пјҢдёәдәҶеҢәеҲҶжҜҸдёҖдёӘдәәпјҢз»ҷжҜҸдёҖдёӘдәәеҸ‘дёҖдёӘдјҡиҜқж ҮиҜҶ(session id), иҜҙзҷҪдәҶе°ұжҳҜдёҖдёӘйҡҸжңәзҡ„еӯ—дёІпјҢжҜҸдёӘдәәж”¶еҲ°зҡ„йғҪдёҚдёҖж ·пјҢ жҜҸж¬ЎеӨ§е®¶еҗ‘жҲ‘еҸ‘иө·HTTPиҜ·жұӮзҡ„ж—¶еҖҷпјҢжҠҠиҝҷдёӘеӯ—з¬ҰдёІз»ҷдёҖе№¶жҚҺиҝҮжқҘпјҢ иҝҷж ·жҲ‘е°ұиғҪеҢәеҲҶејҖи°ҒжҳҜи°ҒдәҶгҖӮ

дҪҶжҳҜиҜ·жұӮйҮҸи¶ҠжқҘи¶ҠеӨ§зҡ„ж—¶еҖҷеҚ•еҸ°жңҚеҠЎеҷЁиӮҜе®ҡжүҝеҸ—дёҚдҪҸе‘ҖпјҢжүҖд»Ҙе°ұжңүдәҶжңҚеҠЎеҷЁзҡ„йӣҶзҫӨжЁЎејҸпјҢйӣҶзҫӨйҮҢйқўжңүжңҚеҠЎеҷЁAе’ҢжңҚеҠЎеҷЁBпјҢиҝҷж ·еҰӮжһңжңҚеҠЎеҷЁAдёҠзҡ„иҜ·жұӮиҪ¬еҸ‘еҲ°жңҚеҠЎеҷЁBзҡ„иҜқпјҢжңҚеҠЎеҷЁе°ұдёҚзҹҘйҒ“иҝҷдёӘsessionеҜ№еә”зҡ„з”ЁжҲ·дҝЎжҒҜдәҶгҖӮеҸҜд»ҘйҮҮз”Ёsession stickyпјҢи®©иҜ·жұӮдёҖзӣҙеҫҖAдёҠйқўеҸ‘пјҢдҪҶжҳҜжІ»ж №дёҚжІ»жң¬гҖӮ

еҹәдәҺжңҚеҠЎеҷЁйӘҢиҜҒж–№ејҸжҡҙйңІзҡ„дёҖдәӣй—®йўҳ

  1. SeesionпјҡжҜҸж¬Ўи®ӨиҜҒз”ЁжҲ·еҸ‘иө·иҜ·жұӮж—¶пјҢжңҚеҠЎеҷЁйңҖиҰҒеҺ»еҲӣе»әдёҖдёӘи®°еҪ•жқҘеӯҳеӮЁдҝЎжҒҜгҖӮеҪ“и¶ҠжқҘи¶ҠеӨҡзҡ„з”ЁжҲ·еҸ‘иҜ·жұӮж—¶пјҢеҶ…еӯҳзҡ„ејҖй”Җд№ҹдјҡдёҚж–ӯеўһеҠ гҖӮ

  2. еҸҜжү©еұ•жҖ§пјҡеңЁжңҚеҠЎз«Ҝзҡ„еҶ…еӯҳдёӯдҪҝз”ЁSeesionеӯҳеӮЁзҷ»еҪ•дҝЎжҒҜпјҢдјҙйҡҸиҖҢжқҘзҡ„жҳҜеҸҜжү©еұ•жҖ§й—®йўҳгҖӮ

  3. CORS(и·Ёеҹҹиө„жәҗе…ұдә«)пјҡеҪ“жҲ‘д»¬йңҖиҰҒи®©ж•°жҚ®и·ЁеӨҡеҸ°з§»еҠЁи®ҫеӨҮдёҠдҪҝз”Ёж—¶пјҢи·Ёеҹҹиө„жәҗзҡ„е…ұдә«дјҡжҳҜдёҖдёӘи®©дәәеӨҙз–јзҡ„й—®йўҳгҖӮеңЁдҪҝз”ЁAjaxжҠ“еҸ–еҸҰдёҖдёӘеҹҹзҡ„иө„жәҗпјҢе°ұеҸҜд»ҘдјҡеҮәзҺ°зҰҒжӯўиҜ·жұӮзҡ„жғ…еҶөгҖӮ

  4. CSRF(и·Ёз«ҷиҜ·жұӮдјӘйҖ )пјҡз”ЁжҲ·еңЁи®ҝй—®й“¶иЎҢзҪ‘з«ҷж—¶пјҢд»–д»¬еҫҲе®№жҳ“еҸ—еҲ°и·Ёз«ҷиҜ·жұӮдјӘйҖ зҡ„ж”»еҮ»пјҢе№¶дё”иғҪеӨҹиў«еҲ©з”Ёе…¶и®ҝй—®е…¶д»–зҡ„зҪ‘з«ҷгҖӮ

еңЁиҝҷдәӣй—®йўҳдёӯпјҢеҸҜжү©еұ•иЎҢжҳҜжңҖзӘҒеҮәзҡ„гҖӮ

Cookie

cookieжҳҜз”ұжңҚеҠЎеҷЁз”ҹжҲҗ,еҸ‘йҖҒз»ҷжөҸи§ҲеҷЁ,жөҸи§ҲеҷЁеҗ§cookieд»ҘkvеҪўејҸдҝқеӯҳеҲ°жң¬ең°жҹҗдёӘзӣ®еҪ•дёӢ,дёӢж¬ЎиҜ·жұӮеҗҢдёҖдёӘзҪ‘з«ҷзҡ„ж—¶еҖҷ,дјҡжҠҠиҜҘcookieеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁгҖӮ

cookieзҡ„з»„жҲҗжңүпјҡеҗҚз§°(key)гҖҒеҖј(value)гҖҒжңүж•Ҳеҹҹ(domain)гҖҒи·Ҝеҫ„(еҹҹзҡ„и·Ҝеҫ„пјҢдёҖиҲ¬и®ҫзҪ®дёәе…ЁеұҖ:вҖқ\вҖқ)гҖҒеӨұж•Ҳж—¶й—ҙгҖҒе®үе…Ёж Үеҝ—(жҢҮе®ҡеҗҺпјҢcookieеҸӘжңүеңЁдҪҝз”ЁSSLиҝһжҺҘж—¶жүҚеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁ(https))гҖӮ
